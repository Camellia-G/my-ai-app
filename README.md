# My AI App

A Python project。

## project struction

```
my-ai-app/
├── src/
│   ├── __init__.py
│   └── main.py
├── .env
├── .gitignore
└── README.md
```

## Tech Stack
Language: Python 3.10+
Version Control: Git&GitHub


## installation

1. git clone https://github.com/your-username/my-ai-app.git
cd my-ai-app
2. create a `.env` file and add your API key
    API_KEY=you_api_key

## Usage
Run the application from the project root：
```bash
python src/main.py
```
